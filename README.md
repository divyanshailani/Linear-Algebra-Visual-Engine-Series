This series started after studying **3Blue1Brown's Essence of Linear Algebra** and asking a simple question:

**Can the hardest ideas in linear algebra be proved by building them as engines?**

This repository is the master landing page for that answer. Each project below has its own source repo, but this page is the fastest way to understand the full arc.

## How This Series Is Built

1. Learn the concept by hand first.
2. Prove it in Python and NumPy.
3. Validate the result with checks, reports, and expected behavior.
4. Rebuild the same idea cinematically in Blender with `bpy`.
5. Publish the logic and the visual proof together.

That workflow is the real point of the series. The render matters, but the proof comes first.
